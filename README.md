# Birthday Scrapbook (GitHub Pages)

A tiny scrapbook-style "year wrapped" website.

## Customize
- Edit `index.html`:
  - Replace the letter text
  - Update numbers like nights, adventures, etc.
- Replace / add photos in `assets/img/`
  - Then update the `<img src="...">` paths in `index.html`

## Local preview
Just open `index.html` in your browser.
